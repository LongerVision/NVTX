For C API:
- First, delete all contents of ../doxygen directory -- that is where the output files go
- In this directory, run "doxgen doxyfile-c"
- The generated output should appear in ../doxygen

For C++ API:
- First, delete all contents of ../doxygen-cpp directory -- that is where the output files go
- In this directory, run "doxgen doxyfile-cpp"
- The generated output should appear in ../doxygen-cpp
